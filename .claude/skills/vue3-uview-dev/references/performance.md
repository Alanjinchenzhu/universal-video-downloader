# 小程序性能优化指南 (uni-app)

## 1. 减少包体积

- **分包加载**: 将不同模块的代码分包。
- **静态资源优化**: 图片使用 WebP 或外挂 CDN。
- **无用代码移除**: 定期清理无用组件、工具库。

## 2. 页面加载优化

- **分批渲染**: 大数据量使用虚拟滚动或分批加载。
- **网络请求**: 合并 API 请求，减少 HTTP 开销。
- **懒加载**: 组件懒加载，避免首次加载过慢。

## 3. 响应式与性能

- **避免深层监听**: `watch` 监听深度嵌套的对象会降低性能。
- **及时销毁**: 页面 `onUnload` 时清理定时器、全局事件监听。
- **setData 最小化**: 尽量减少 `ref` 频繁更新引起的小程序通讯开销。
